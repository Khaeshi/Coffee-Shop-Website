  ## Description

  This website was built to help Julian Garcia's business, Gwampits, shine online! It's designed to be the perfect digital storefront, showcasing their Coffee products and making it super easy for potential customers to find them and get in touch. We focused on creating a user-friendly gallery website experience that reflects Julian Garcia's brand and helps them grow their business.


  ## Features

  *   Responsive design (works on all devices)
  *   Interactive photo galleries
  *   Contact form
  *   Blog section for news and updates
  *   Social media integration

  ## Technologies Used

  React.js, Figma, Tailwind


  ## Installation 

  1.  Clone the repository: `git clone [repository URL]`
  2.  Install dependencies: `npm install`
  3.  Run the application: `npm start`
